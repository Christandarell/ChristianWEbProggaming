HTML & CSS Quick Reference Guide

HTML Tags

  • <html> - It determines the starting point of a document written in HTML..
  • <head> - It includes meta information regarding the file itself..
  • <title> - Gives the text of the document's title, which is displayed in the browser's topic bar or window.
  • <h1> to <h6> - Heading tags: largest to shortest.
  • <p> - Defines a paragraph.
  • <a> - Defines a hyperlink.
  • <img> - Embeds an image.
  • <ul> - Defines an unordered list.
  • <li> - Defines a list item.

CSS Properties

  • color - Sets the text color.
  • font-family - Sets the font family for text.
  • font-size - Sets the font size for text.
  • background-color - Sets the background color of an element.
  • margin - Sets the margin around an element.
  • padding - Sets the padding inside an element.
  • border - Sets the border of an element.
  • text-align - Sets the alignment of text.
